# 前端启动说明

## 安装依赖

```bash
cd frontend
npm install
```

## 启动开发服务器

```bash
npm run dev
```

默认地址：`http://localhost:5173`

代理规则：

- `/api/*` -> `http://localhost:5555`

## 环境说明

当前脚手架已做 Node 12 兼容（Vite 2 系列）。  
如果后续升级到 Node 18+，可再升级 Vite 与相关依赖。

## 登录验证

打开首页后会先跳转到 `/login`。  
建议先在项目根目录执行 `./db/load_full_seed.sh` 导入完整样本数据。  
默认测试账号：

- 用户名：`admin`
- 密码：`admin123`

登录成功后进入系统主页面；点击右上角“退出登录”会清除前端会话并返回登录页。

## 页面说明（当前）

- `/`：专业决策看板（就业率/对口率/平均匹配分/技能缺口）
- `/majors`：学院/专业/专业画像管理
- `/jobs`：企业/岗位管理
- `/students`：学生/就业记录管理
- `/match`：单条匹配、批量匹配、历史结果解释查询
