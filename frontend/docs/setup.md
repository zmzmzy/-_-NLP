# 前端启动说明

本文件聚焦前端细节；完整项目启动说明请先看根目录 `README.md`。

## 安装依赖

```bash
cd /home/roamer/graduation_proj_0.1/frontend
npm install
```

## 启动开发服务器

```bash
npm run dev
```

默认地址：`https://127.0.0.1:5173`

## 开发代理

- 默认开发地址：`https://127.0.0.1:5173`
- 可临时切换 HTTP：`npm run dev:http`
- `/api/*` 代理到 `http://127.0.0.1:5555`
- 配置文件：`frontend/vite.config.js`

## 登录与会话说明

- 打开页面会先跳转 `/login`
- 系统无默认账号，请先注册
- 同一浏览器里切换管理员/学生账号会共享 Cookie，会出现权限被覆盖
- 联调双角色时建议使用不同浏览器或无痕窗口

## 页面说明（当前）

- `/`：决策看板（专业聚合评分与排名）
- `/majors`：学院/专业/专业画像管理
- `/jobs`：企业/岗位管理
- `/students`：学生与就业记录管理
- `/match`：匹配分析（单条、批量、重算、结果查询）
- `/profile`：个人主页（学生资料绑定、就业提交；管理员审核）
